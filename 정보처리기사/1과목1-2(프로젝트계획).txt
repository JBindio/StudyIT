1.프로젝트 계획
	1)프로젝트 관리
	- 프로젝트 핵심 관리대상(3P)
		- 사람, 문제, 프로세스
	- PMBOK
		- PMP 자격증 기술서
		- PM(프로젝트 매니저)
		- 5단계 : 착수, 계획, 실행, 통제, 종료
	
	2)개발비용 산정
		- 하향식
			- 전문가 기법
			- 델파이 기법 : 여러명의 전문가
		- 상향식
			- LOC : 추정LOC (낙관치 + (4*중간치) + 비관치)/6
			- 단계별 인원수
		- 수학적
			- COCOMO
				- 조직형(Organic) : 5만 이하
				- 반분리형(Semidetached) : 30만 이하
				- 내장형(Embedded) : 30만 이상
			- Putnam
				- Rayleigh-Norden 곡선
				- SLIM 자동화 추정도구
			- 기능점수기법(FP)
				- 기능의 개수를 기준으로 규모 측정
				- ESTIMACS
	3)개발일정 산정
		- 소프트웨어 개발일정 계획
		- WBS : 작업분해도(세분화)
		- PERT/CPM 네트워크 : 선후행 관계 고려 -> 우선순위 결정
			- 임계경로 : 프로젝트를 끝내기 위해 필요한 최소 소요시간
				- 빠른 착수일 : 최소기간
				- 늦은 착수일 : 임계경로 - 최소경로 + 빠른착수일
				- 여유기간 : 늦은착수일 - 빠른착수일
		- 간트차트
