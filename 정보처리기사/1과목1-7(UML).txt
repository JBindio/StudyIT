1과목 1-7
1.UML★ : 그림 그리는 언어
	1) UML 특징
		- 가시화 언어 : 눈에 보이는
		- 명세화 언어 : 명세화
		- 구축 언어 : 변환, 구축
		- 문서화 언어 : 유지, 보수
	
	2) UML 구성요소
		- 사물(Things)
			- 구조 사물
			- 행동 사물
			- 그룹 사물
			- 주해 사물
		- 관계(Relationship)
			- 일반화관계(Generalization)
				- 상속관계(Inheritance)
			- 연관관계(Relationship)
				- 지속적인 관련된 관계(사람 -> 휴대폰)
			- 의존관계(Dependency)
				- 짧은 관련된 관계(사람 -> 색연필)
			- 실체화관계(Realization)
				- 재정의(오버라이딩)
				- 오퍼레이션
				- 같은 버튼 -> 속도조절, 볼륨조절 재정의
			- 집합관계
				- 집약관계(Aggregation)
					- 서로 독립적인 관계
				- 합성관계(Composition)
					- 서로 필수적인 관계
		- 다이어그램
			- 구조 다이어그램 : 변하지 않는 틀
				- ★클래스 다이어그램 : 클래스 - 클래스 관계
				- 객체 다이어그램 : 객체 - 객체 관계
				- 컴포넌트 다이어그램
				- 배치 다이어그램 : 위치
				- 복합체 다이어그램 : 복합구조
				- 패키지 다이어그램 : 그룹화, 패키지
			- 행위 다이어그램 : 움직이고 변하는 틀
				- ★유즈케이스 다이어그램 : 수행하는 행동, 기능 분석
					- Actor(사용자), UseCase(기능), System(시스템)
					- 연관관계(Association) : 상호작용,실선
					- 포함관계(Include) : 반드시 실행되야 하는 것, 점선
					- 확장관계(Extend)  : 특정조건
					- 일반화관계(Generalization) : 추상화
				- ★시퀀스 다이어그램 : 순서, 상호작용
				- 커뮤니케이션 다이어그램
				- 상태 다이어그램
				- 활동 다이어그램
				- 타이밍 다이어그램
	
	3)주요 다이어그램
		- 구조 다이어그램
			- 클래스 다이어그램
