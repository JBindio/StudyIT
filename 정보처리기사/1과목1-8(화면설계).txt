1.화면설계
	1)UI(User Interface) : 사용자 인터페이스
		- UI유형
			- CLI : 키보드로 명령어 입력
			- GUI : 그래픽과 텍스트, 마우스
			- AUI : 임베디드 사운드
			- NUI : 자연스러운 움직임
		- UI설계
			- 기능적 요구사항
				- 입력,출력,데이터,연산
			- 비기능적 요구사항
				- 품질, 기술, 시스템, 비용, 일정
		- UI설계 원칙★
			- 직관성 : 쉽게 이해, 사용
			- 유효성 : 정확한 목적달성
			- 학습성 : 쉽게 배우고 익힘
			- 유연성 : 요구사항 수용, 오류 최소화
		- UI설계 도구
			- 와이어프레임 : 대충 그린 그림(기획자 -> 디자이너)
			- 스토리보드 : 최종 결과 예상(디자이너 -> 개발자)
			- 프로토타입 : 동적인 시제품, 단기간, 구현가능
			- 목업 : 정적인 모형, 구현불가
			- 유스케이스
	2)UX(User Experience) : 사용자의 경험
