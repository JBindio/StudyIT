마무리특강 3강
1.서버 환경 구축
	1)Web서버 : 정적
	2)Was서버 : 동적
	3)DB서버
		- 공용데이터
		- 저장데이터
		- 통합데이터
		- 운영데이터
	4)File서버 : 공통 파일서버
	5)Load Balancer
	6)CDN : 분산 저장 서버
	
2.개발 소프트웨어 환경
	1) 시스템 소프트웨어
		- 운영체제
		- JVM
		- Web서버
		- Was서버
		- DBMS서버
	2) 개발 소프트웨어
		- IDE(통합개발도구)
		- 협업도구 : 게시판, 메신저
		- 버전관리도구
			- 공유폴더방식 : CVS, SVN,
			- 클라이언트/서버방식
			- 분산저장소 방식
			- 버전관리 주요 용어
		- 형상관리도구 : 변경사항을 버전별로 관리하는 도구
			- 형상관리절차
				- 식별, 통제★, 감사, 기록
		- 빌드도구 : 소스코드 -> 소프트웨어
			- 빌드 자동화 도구
				- Make, Ant, Maven, Jenkins, Gradle
				
3.개발 프레임워크 : 개발에 공통적으로 사용되는 여러가지 기능들을 제공해주는 소프트웨어
	1) 프레임워크 특징
		- 모듈화
		- 재사용성
		- 확장성
			- 다형성 : 오버로딩, 오버라이딩
		- 제어의 역흐름
	2) 라이브러리 : 빈번에게 사용되는 리소스를 모아둔것
	3) API : 일종의 소프트웨어 인터페이스(완제품)
